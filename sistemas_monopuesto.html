Módulo: Sistemas Operativos Monopuesto

En este módulo he aprendido a instalar, configurar y gestionar sistemas operativos utilizados en un solo puesto de trabajo. A continuación, te presento un resumen de las competencias adquiridas durante este curso.

Competencias Adquiridas

Instalación de Sistemas Operativos

He aprendido a instalar correctamente sistemas operativos como Windows y Linux desde cero, utilizando tanto dispositivos físicos como máquinas virtuales.

Configuración del Sistema

Configuré parámetros esenciales como la resolución de pantalla, idioma, región, nombre del equipo, cuentas de usuario y administración de energía.

Instalación de Software

Aprendí a instalar aplicaciones esenciales como navegadores, paquetes ofimáticos, antivirus y herramientas específicas para cada entorno laboral.

Gestión de Usuarios y Permisos

Configuré cuentas de usuario, grupos, y asigné permisos adecuados para garantizar la seguridad y privacidad dentro del sistema operativo.

Copias de Seguridad y Restauración

Aprendí a realizar copias de seguridad automáticas y restauraciones del sistema ante fallos o pérdida de información.

Mantenimiento del Sistema

Realicé tareas de limpieza, actualización del sistema, desfragmentación de disco y revisión de rendimiento para garantizar un funcionamiento óptimo.
