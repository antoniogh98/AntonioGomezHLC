Ofimática

Dominando herramientas ofimáticas para la productividad personal y profesional.

¿Qué he aprendido en este módulo?

En el módulo de Ofimática he aprendido a utilizar con soltura las principales aplicaciones de oficina, como procesadores de texto, hojas de cálculo y presentaciones. Además, he desarrollado habilidades para crear documentos profesionales, organizar datos de forma eficiente y diseñar presentaciones visualmente atractivas.

Competencias Desarrolladas

  • Dominio de Microsoft Word y LibreOffice Writer para elaborar documentos formales.
  • Uso avanzado de Microsoft Excel y LibreOffice Calc para realizar cálculos, gráficos y análisis de datos.
  • Creación de presentaciones dinámicas con Microsoft PowerPoint y LibreOffice Impress.
  • Aplicación de formatos, estilos y estructuras en documentos empresariales.
  • Gestión y automatización de tareas básicas con fórmulas y funciones.

Herramientas Utilizadas

Microsoft Word

Microsoft Excel

Microsoft PowerPoint
