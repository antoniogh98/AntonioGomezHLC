Aplicaciones Web

Diseño y desarrollo de páginas web usando HTML, CSS, JavaScript y frameworks modernos.

¿Qué he aprendido en este módulo?

En este módulo he aprendido a desarrollar sitios web estáticos y dinámicos, trabajando tanto el diseño visual como la estructura del contenido. También he comprendido la importancia de la accesibilidad, el responsive design y el uso de frameworks como Bootstrap.

Competencias Desarrolladas

  • Diseño y maquetación de páginas web con HTML5 y CSS3.
  • Uso de Bootstrap 5 para diseño responsive y componentes UI.
  • Incorporación de scripts con JavaScript para interactividad.
  • Organización de proyectos web en Visual Studio Code.
  • Subida de páginas web a un servidor local usando XAMPP.
